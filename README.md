## 🚀 About The Project

Book Notes is a modern full-stack web application where users can explore books, save them to their collection, give ratings, write personal notes, and manage their reading journey.

This project was built to practice real-world backend development, authentication systems, API integration, and database management.

---

## ✨ Features

- 🔍 Search books using external book APIs  
- 📚 Add books to your personal library  
- ⭐ Rate books based on your reading experience  
- 📝 Write notes / reviews for each book  
- 👤 User Authentication & Authorization  
- 🔐 Secure Login System  
- ❤️ Favorite books / reading wishlist  
- 📅 Track reading dates and progress  
- 🌐 Fully responsive UI
